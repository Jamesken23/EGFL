# EGFL
This repo is a paper on Python implementation: **Vulnerability detection in smart contracts via enhanced graph feature learning.** 

# Required Packages
> - Python 3.7+
> - Keras 2.3.1
> - numpy 1.19.5
> - scikit-learn 1.3.0
> - tensorflow 1.15.0


# Running
To run the program, please use this command: python main_run.py
